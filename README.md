## 📖 Detailed Task Descriptions

### 1. Netflix Data Cleaning & Formatting (`Data-Analyst-Task-1`)
* Cleaned and normalized the raw Netflix movie data using Pandas.
* Implemented Python scripts to automate Excel layout adjustments, font stylings, borders, cell sizes, and summary sheets.
* **Deliverables**: Python scripts (`format_excel.py`, `step1_analysis.py`) and formatted spreadsheet files.

### 2. Superstore Profitability Analysis (`Task_2_Superstore-Profitability-Analysis`)
* Conducted a detailed diagnostic analysis on superstore transactional data to locate profit leakages.
* **Deliverables**: Tableau packaged workbook (`Superstore_Profit_Analysis.twbx`) and a PDF printout of the findings dashboard.

### 3. Interactive Auto Sales Dashboard (`Task-3-Dashboard-Design`)
* Created an interactive executive dashboard visualizing global automobile sales trends.
* **Deliverables**: Tableau workbook (`Task 3- The Interactive Dashboard.twb`), summary PowerPoint presentation slides, and dashboard mockups.

### 4. SQL for Data Analysis (`Task-4-SQL-for-Data-Analysis`)
* Wrote production SQL queries covering aggregates, window functions, and joins to calculate customer cohort sizes, purchase frequency, and product popularity.
* **Deliverables**: SQL script file (`task4_queries.sql`) and a detailed query explanation report.

### 5. Web Product Scraper & EDA (`Task-5-Exploratory-Data-Analysis`)
* Programmed a BeautifulSoup python scraper to extract product listings, pricing, and ratings.
* **Deliverables**: Scraper code script (`scraper.py`) and output product CSV databases.

### 6. Sales Trend Analysis Using Aggregations (`TASK-6-Sales-Trend-Analysis-Using-Aggregations`)
* Processed massive scale transactional records (`terror.csv`) to compute rolling metrics and trend aggregations.
* **Deliverables**: Jupyter Notebook (`deliverable.pynb`) detailing aggregation code and analysis.

### 7. Time Series Forecasting (`Task-7-Time-Series-Analysis`)
* Applied statistical models (ARIMA) to forecast time-series data using `AirPassengers.csv` tracking metrics.
* **Deliverables**: Jupyter Notebook walkthrough (`Task-7-Time-Series-Analysis.ipynb`) outlining trend analysis and forecasts.

### 8. Sales Executive Dashboard Design (`Task-8-dashboard-design`)
* Developed high-fidelity visualization dashboards in Tableau to display regional executive sales details.
* **Deliverables**: Packaged Tableau worksheets and dashboard screenshots.

### 9. Retail Business Performance & Profitability (`Retail-Business-Performance-Profitability-Analysis`)
* Engineered SQL database queries to calculate sales profitability and business performance indicators.
* Built analytical models in Python (`retail_project.py`) to process store transactions and generate automated PDF profitability reports.
* **Deliverables**: Python script (`retail_project.py`), SQL queries (`Project_Queries.sql`), and executive analysis report (`Retail_Analysis_Report.pdf`).
