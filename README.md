# Task 3: Interactive Sales Dashboard

This repository contains the completed project for the Data Analyst Internship Task 3, which focuses on designing an interactive dashboard for business stakeholders.

## 📝 Objective

The goal was to build a fully interactive dashboard using Tableau to track key business metrics from the "Auto Sales" dataset. The dashboard provides a high-level view of performance while also allowing users to filter and drill down for specific insights.

## 🛠️ Tools Used

* **Tableau**: Used for all data visualization and dashboard creation.
* **PowerPoint**: Used to create the final summary presentation.

## 💡 Key Insights

This dashboard answers several key business questions:

* **KPIs at a Glance**: The dashboard tracks high-level metrics like **Total Sales** and **Total Quantity Ordered**.
* **Top-Performing Products**: The **'Classic Cars'** product line is the most significant driver of sales.
* **Primary Market**: The **USA** is the largest and most critical market by sales volume.
* **Full Interactivity**: The dashboard is interactive. Clicking a country on the map or a product on the bar chart filters all other charts to show specific insights for that selection.
