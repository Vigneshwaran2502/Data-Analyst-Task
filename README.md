# Superstore Profitability Analysis

This is a data visualization project for Task 2, "Data Visualization and Storytelling."

## Objective
To analyze the Superstore sales data to find a compelling business story. This dashboard focuses on identifying *what* products are unprofitable, *where* the losses are, and *why* it's happening.

## Tools Used
* Tableau

## Key Findings
The analysis shows that while the company is profitable, it is losing a lot of money on **'Tables'** and **'Bookcases'**. These losses are concentrated in states like **Texas** and **Illinois**. The scatter plot proves this is because these items are being sold at very **high discounts (30-50%)**, which is directly causing the losses.
